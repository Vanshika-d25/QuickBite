# 🍔 QuickBite

QuickBite is a modern React-based food ordering web application that allows users to explore restaurants, view menus, and enjoy a smooth browsing experience.

---

## 🚀 Features

* 🔍 Browse restaurants dynamically
* 📋 View detailed restaurant menus
* ⚡ Fast and responsive UI
* 🌐 Online/Offline status detection
* 🧩 Modular component-based architecture
* 💡 Clean and scalable code structure

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS
* **Build Tool:** Parcel
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/Vanshika-d25/QuickBite.git
```

2. Navigate to the project folder

```
cd QuickBite
```

3. Install dependencies

```
npm install
```

4. Run the app

```
npm start
```
